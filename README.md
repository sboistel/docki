# Mkdocs

## Mkdocs local
How to localy check any changes with `mkdocs` command ? 

From the root of docki repo :

```bash
mkdocs serve
```

## Mkdocs Build
How to build your changes into the `site` directory ?

From the root of docki repo :

```bash
mkdocs build
```

## Mkdocs Publish
How to publishing changes into the `site` directory ?

From the root of docki repo :

```bash
mkdocs gh-deploy
```

:warning: If U've don't commit any changes, take care on it first
