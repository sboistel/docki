Guacamole

Apache Guacamole is a clientless remote desktop gateway. \ It supports standard protocols like VNC, RDP, and SSH.

We call it clientless because no plugins or client software are required. \ Thanks to HTML5, once Guacamole is installed on a server, all you need to access your desktops is a web browser.

Server

Requiere depedences

  • APT:DEB/Ubuntu > apt install libcairo2-dev libtool-bin libossp-uuid-dev

Optional depedences

  • APT:DEB/Ubuntu > apt install libavcodec-dev libavformat-dev libavutil-dev libswscale-dev freerdp2-dev libpango1.0-dev libssh2-1-dev libtelnet-dev libvncserver-dev libwebsockets-dev libpulse-dev libssl-dev libvorbis-dev libwebp-dev

Obtain code

  • APT:DEB/Ubuntu bash apt install git

Donwload the guacamole server repository:

git clone git://github.com/apache/guacamole-server.git

Build process

cd guacamole-server
autoreconf -fi

./configure --with-init-dir=/etc/init.d --enable-allow-freerdp-snapshots
make

Installation

make install
ldconfig

Guest agent

Requierement

  • APT:DEB/Ubuntu > apt install git maven
  • RPM:Fedora/Centos/Rhel: > yum install git maven

Donwload the guacamole client repository

git clone git://github.com/apache/guacamole-client.git
mvn package
